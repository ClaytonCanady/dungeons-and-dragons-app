# dungeons-and-dragons-app
### Clayton Candy
This application uses react and react-router to access the D&D 5th edition API found at http://www.dnd5eapi.co/

- Over 300 spells
- Over 300 monsters
- 15 in Game Conditions

- React
- React Bootstrap
- React Router
